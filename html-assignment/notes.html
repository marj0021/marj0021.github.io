HTML Tags

These are 5 HTML tags I have learnt and their description:

Image
The tag <img> lets you add images and that is crucial because it helps to deliver your product as informative as possible.
Organized List
the tag <ol> offers you the option to add a list which the order in it is important
Headers
HTML headings are defined with the <h1> to <h6> tags. and it's from the most important to the least important.
Paragraphs
The HTML tag <p> element defines a paragraph. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.
Button
The "button" tag defines a clickable button. Inside a <button > element you can put text (and tags like <i >,<b> , <strong>, <br>, <img>, etc.)
